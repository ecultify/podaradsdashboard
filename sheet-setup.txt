Google Sheet → dashboard (CSV)

1. Create a sheet with two columns (row 1 can be headers: variable | value):

   total_amount     <your total budget number>
   amount_spent     <optional; if omitted, Meta spend × multiplier is used>
   link_clicks      <optional; if omitted, Meta link clicks are used>

2. Balance on the dashboard = total_amount − effective spend.
   Effective spend = amount_spent from the sheet (or env), else Meta API spend × NEXT_PUBLIC_SPEND_MULTIPLIER.

3. Share the sheet: “Anyone with the link” can View (required for the export URL).

4. In Vercel (or .env.local), set either:
   GOOGLE_SHEET_ID=<id from the spreadsheet URL>
   GOOGLE_SHEET_GID=0
   (GID = tab id; first tab is usually 0)

   Or paste the full CSV export URL:
   GOOGLE_SHEET_CSV_URL=https://docs.google.com/spreadsheets/d/<SHEET_ID>/export?format=csv&gid=0

5. Redeploy after changing env vars.

Priority for each number: MANUAL_* env variables → Sheet → public/manual-values.json → Meta (where applicable).

Optional env:
MANUAL_TOTAL_AMOUNT=
MANUAL_AMOUNT_SPENT=
MANUAL_LINK_CLICKS=
MANUAL_BALANCE_LEFT=   (if set, overrides computed balance)
